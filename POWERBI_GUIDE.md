# Power BI Dashboard Guide

Follow these steps to visualize the data generated by the ETL pipeline.

## Step 1: Load Data
1. Open **Power BI Desktop**.
2. Click on **Get Data** (Home ribbon) -> **Text/CSV**.
3. Browse to your project folder and select **`cleaned_jobs_for_dashboard.csv`**.
4. In the preview window, click **Transform Data** (not just Load).

## Step 2: Prepare Data (Power Query)
*We need to process the 'skills_required' column to count individual skills, as they are currently in a single cell (e.g., "Python, SQL").*

1. **Split Skills References**:
   - Select the **`skills_required`** column.
   - Go to the **Transform** tab (or right-click the column header).
   - Select **Split Column** -> **By Delimiter**.
   - Ensure "Comma" is selected.
   - Expand **Advanced Options**.
   - Select **Split into Rows** (This is crucial to count each skill separately).
   - Click **OK**.
2. **Close & Apply**:
   - Click **Close & Apply** in the top-left corner to load the transformed data into the report view.

## Step 3: Create Visualizations

### 1. Top 5 Demanded Skills
- Select the **Clustered Bar Chart** icon.
- Drag **`skills_required`** to the **Y-axis**.
- Drag **`job_id`** to the **X-axis** (it will default to "Count of job_id").
- In the Filters pane (on the visual), expand `skills_required` and set Filter Type to **Top N**.
  - Show items: Top **5**.
  - By value: Drag **`job_id`** (Count) into the field.
  - Click **Apply filter**.

### 2. Average Salary
- Select the **Card** visual (123 icon).
- Drag **`min_salary`** or **`max_salary`** to the Fields.
- Click the small arrow next to the field name and change "Sum" to **Average**.
- (Optional) Rename "Average of min_salary" to "Avg Min Salary (LPA)".

### 3. Job Postings by Location
- Select the **Map** visual or **Pie Chart**.
- Drag **`location`** to the **Location/Legend** field.
- Drag **`job_id`** to the **Bubble size/Values** field (Count).
